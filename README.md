This project is a historical financial statement analysis of Nykaa (FSN E-Commerce Ventures Ltd.), developed using Microsoft Excel. The model covers the company's financial performance from FY2018 to FY2025–26, including LTM data, and links all three financial statements to provide a complete view of the business.

The analysis includes:

-Historical Income Statement, Balance Sheet, and Cash Flow Statement analysis.

-Comprehensive ratio analysis covering profitability, liquidity, efficiency, leverage, and valuation metrics.

-Trend analysis of key financial line items.

-Basic financial forecasting using the RRI function to estimate sustainable growth rates.

-Statistical analysis using the arithmetic mean, geometric mean, and standard deviation to evaluate historical performance, growth consistency, and financial variability.

This model demonstrates practical financial modeling skills, financial statement analysis, ratio analysis, and fundamental forecasting techniques commonly used in equity research, investment analysis, and corporate finance.
